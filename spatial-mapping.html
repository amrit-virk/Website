Spatial Mapping System

A ToF-based 3D scanning system that maps environments in real time.

Overview

The Spatial Mapping System uses a time-of-flight (ToF) distance sensor mounted on a pan-tilt system to gather 3D positional data. Data is collected in real-time and processed to generate a point cloud representation of the scanned area.

Tools & Technologies

  • VL53L1X ToF Sensor
  • Arduino Nano
  • Servo Motors (Pan/Tilt)
  • Python (Visualization with Open3D)
  • Serial Communication

Challenges

Ensuring accurate timing between sensor readings and motor positions was critical. We implemented a feedback system and synchronized scan rates to improve resolution and reduce drift.
